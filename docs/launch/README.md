## Suggested order (≈ 1 day)

1. **Morning** — Show HN (US Pacific 8am Tue/Wed/Thu)
2. **+30 min** — English Twitter thread, link the HN post in tweet 5 as PS
3. **Lunch** — Reddit r/macapps
4. **Afternoon** — Product Hunt scheduled for next 12:01am Pacific
5. **Evening** — Chinese Twitter thread + V2EX + 即刻

The Show HN and Reddit posts are the only ones that can't be deleted/rescheduled
once live, so post those when you can stay around for 2–3 hours of replies.

## Reply hygiene

The first ~10 replies on any post pull the algorithm. Keep a tab open and
respond to every reply for the first hour:
- Compliment → "thanks 🙏, what theme do you want more of?"
- Bug → file the issue in front of them, link the issue back
- Feature ask → either "good idea, opened #N" or "deliberately scoped out, here's why"

Don't argue. Don't auto-defend. Don't drop links.
