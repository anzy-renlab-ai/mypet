## Text (first comment)

```
I'm a Claude Code user and my $20/mo subscription was sitting underused outside
of work hours. So I built a fluffy desktop cat for macOS that you hover over
for ~1 second; it chomps one `claude -p` call and a speech bubble pops with
one of six things, weighted: a Claude Code tip, a prompt to type into Claude
Code right now, a one-line tech-news headline, a "today I learned", a
programmer joke, or a programmer haiku.

No background work — zero CPU and zero network when you're not hovering. No
separate Anthropic API key — it spends your existing Claude Code quota by
shelling out to the local `claude` CLI. The cat lives in the bottom-right of
your screen, drags around, snaps to edges. Click the tip bubble to copy it.

It's MIT, 87 tests, SwiftUI + AppKit, macOS 13+. README has a demo GIF.

Built it mostly to scratch my own itch (and to see what a single-binary
Tamagotchi-for-LLMs would look like). Curious if anyone else's subscription
sits idle the same way.
```
