mypet 🐱

A fluffy desktop cat that eats your Claude Code tokens.

Get it on GitHub → Install

You pay for Claude Code anyway. The little cat spends your subscription quota — no separate API key, no server, no telemetry. When you're not feeding it, it costs zero CPU and zero network. When you do feed it, you get one cute interruption and one tiny morsel of useful information.

It's a screensaver that pays rent. 🐈

How it works

STEP 1

Cursor near cat

A wobbly little Claude-cookie 🍪 follows your mouse — visual feedback that the cat sees you.

STEP 2

Double-click cat

Single clicks pass through to the app underneath. Only a double-click triggers feeding.

STEP 3

Cat chomps + tip pops

Shells out to your local claude CLI. Reply lands in a tiny speech bubble.

STEP 4

Click bubble to keep

Single-click copies the tip to your clipboard and dismisses. Or leave it — auto-dismiss in 10 min.

Install

Requirements: macOS 13+, the Claude Code CLI on your PATH.

git clone https://github.com/anzy-renlab-ai/mypet
cd mypet
swift run mypet

First launch shows a tiny onboarding wizard. The cat lives in the bottom-right of your primary display. Click 🐾 in the menu bar for Feed now / Snap to edge / Launch at login.
